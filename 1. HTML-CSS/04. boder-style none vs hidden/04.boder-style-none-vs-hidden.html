Como ya sabemos, con border-collapse: collapse; unimos los bordes de las celdas. De hecho, aunque añadamos border-spacing: 50px; el borde se sigue viendo igual. Podéis probarlo.

Si tenemos una celda con un borde visible (como las celdas 2, 6, 8 o 12) y queremos que una celda adyacente no tenga borde debemos indicarlo con border-style: hidden;. Esta instrucción prevalece sobre el borde visible de las celdas que lo tienen.

Si ponemos border-style: none;, entonces el borde visible de las celdas adyacentes prevalece sobre esa border-style: none; y el borde se ve, como en el caso de la celda 9.
